# Description

First stab on Angular 2 based on starter kit featuring the official Tour of Heroes [tutorial](https://angular.io/docs/ts/latest/tutorial/).

# Requirements

* Node.js >= 6.x
* NPM >= 5.x
* Angular CLI >= 1.x

# Getting started

* clone this repo
* open powershell and cd into cloned dir
* npm install
* ng serve
